HTML5 Tags Video, Audio, Canvas

Video tag operates almost exactly like an Image tag, wherein the source must be defined as well as the size. Additionally, you can define if the video control options appear, if the video should autoplay and many other features.

Audio tag operates almost identically to Video, save it plays audio instead of video.

Putting text in between the opening and closing Video/Audio tags is text that will be displayed if the browser is incapable of displaying the video/audio due to compatability or other issues.
